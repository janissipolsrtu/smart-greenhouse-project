# Gudrās siltumnīcas sistēma tomātu audzēšanai

## Par projektu

Šis projekts tiek izstrādāts bakalaura darba ietvaros.

## Akadēmiskā informācija

- **Autors:** Jānis Sīpols  
- **Studiju programma:** Rīgas Tehniskās universitātes Datorzinātnes, informācijas tehnoloģijas un enerģētikas fakultāte  
- **Bakalaura darba tēma:** *Gudrās siltumnīcas sistēmas izstrāde tomātu audzēšanai*  

## Projekta apraksts

Projekta mērķis ir izstrādāt gudrās siltumnīcas sistēmu tomātu audzēšanai.

Sistēma tiek izstrādāta, izmantojot minimāli darbspējīgā produkta (MVP) pieeju, nodrošinot būtiskāko funkcionalitāti, kas nepieciešama sistēmas pamatdarbībai.

Risinājums nodrošinās vismaz obligāti nepieciešamās funkcijas siltumnīcas uzraudzībai un kontrolei, veidojot pamatu turpmākai sistēmas attīstībai un paplašināšanai.

##  Galvenie uzdevumi

- Izstrādāt funkcionējošu gudrās siltumnīcas sistēmas prototipu  
- Ieviest būtiskākās uzraudzības un kontroles funkcijas  
- Nodrošināt sistēmas stabilu pamatdarbību  
- Izveidot paplašināmu pamatu turpmākai attīstībai
